## What is Strawberry Notes?

A simple, self-hostable notes app:

- Rich-text editor (headings, lists, checklists, images).
- Folders, tags, pinning, soft-delete "trash".
- Full-text search across every note you own.
- Markdown export/import.
- Installable as a PWA; reads work offline.
- Dark/light themes with six accent colours.

It is **multi-user** (each deployment supports many accounts) but **not multi-tenant** (there are no organisations, no shared notes between users in v1).
