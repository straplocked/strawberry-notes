## One-Paragraph Summary

Strawberry Notes is an MIT-licensed, self-hostable, multi-user Markdown / rich-text notes app. It runs as a single Next.js container plus Postgres on a server you control — no SaaS dependency, no telemetry, no lock-in. v1 covers the notebook core (folders, tags, search, images, export/import, PWA, theming) and draws the line there: no sharing, no collaboration, no org model. The product exists to be *boringly good at being a notebook that you own*.
